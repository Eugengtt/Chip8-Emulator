# 👾 CHIP‑8 Emulator (Windows Console)

This is a CHIP‑8 emulator written entirely in C++, running directly inside the Windows Command Prompt.  
It uses character‑based rendering to display graphics smoothly without any external visual libraries.

---

## ✨ Features

- **Automatic Game Menu** – scans the current folder and lists all `.ch8` games  
- **Instant Restart** – press `P` at any moment to restart the current game  
- **Optimized Terminal Graphics** – uses `▀`, `▄`, `█` characters to double vertical resolution and reduce flicker  
- **Standalone Build** – runs immediately without installing anything  

---

## 🕹️ Controls

| CHIP‑8 | PC Keyboard |
|--------|-------------|
| 1 2 3 C | 1 2 3 4 |
| 4 5 6 D | Q W E R |
| 7 8 9 E | A S D F |
| A 0 B F | Z X C V |

- `P` – Restart game  
- `ESC` – Exit emulator  

---

## 🚀 How to Run

1. Download **chip8.zip** from this repository  
2. Extract the archive into a folder  
3. Place your `.ch8` ROMs in the same folder  
4. Run **chip8.exe**  
5. Select a game by typing its number in the menu  

**Tip:**  
If the window looks too small, right‑click the console title bar → **Properties** → **Font**, and choose a larger size (36, 48, 72).
